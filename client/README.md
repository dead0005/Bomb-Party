# Bomb Party — Frontend

This folder contains the React frontend for the Bomb Party multiplayer game.

## Getting started

```bash
cd client
bun install
bun run dev
```

## Build

```bash
bun run build
```

## Test

```bash
bun run test
```

## Overview

- Frontend is built with React 19, TypeScript, Vite, and Tailwind CSS.
- Uses Socket.io client for real-time room state and gameplay updates.
- Includes dynamic room creation, chat, avatar selection, audio controls, and game settings.
